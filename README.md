# fluxo de Caixa - Consultoria Financeira
 focada em solucao estratégica para gestão de fluxo de caixa, planejamento, redução de custos e otimização de lucros para negócios

 ## Funcionalidades Planejadas
 * **Painel Institucional:** Apresentação dos serviços de consultoria.
 * **Simulador de Caixa:** Ferramenta interativa para potenciais clientes.
 * **Agendamento:** Formulário integrado para solicitação de diagnósticos financeiros.

 ## Tecnologias que serão utilizadas 
* HTML5 / CSS3 / JavaScript
* Git e GitHub para controle de versão
